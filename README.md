# 🌐 Portfolio Website - Anurag Choudhary

## 📌 Overview
This is a personal portfolio website created using HTML and CSS. The project showcases my education, skills, and contact information in a clean and responsive layout.

## 🚀 Features
- Responsive design
- Clean and modern UI
- Separate pages for Education, Skills, and Contact
- Navigation bar for easy access
- Use of icons for better UI

## 🛠️ Technologies Used
- HTML5
- CSS3
- Phosphor Icons
